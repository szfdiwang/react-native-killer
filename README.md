# react-native-killer

A killer for react-native back to desktop

## Installation


```sh
npm install react-native-killer
yarn add react-native-killer
```


## Usage


```js
import { exitApp } from 'react-native-killer';

// Call the method when needed to kill the process

exitApp();
```
